## Getting Started

First, run the development server:

```bash
npm i
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

for building:
``` bash
npm run build
```

## Folder structure
- src/components (contains all the components used in the main page)
- src/hooks (contains usethrottle hook)
- queries (contains api calls)

## Styling
This project uses tailwind for styling
